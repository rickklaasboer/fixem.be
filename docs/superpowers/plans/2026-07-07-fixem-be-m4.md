# fixem.be M4 Implementation Plan — Threads, Instagram, TikTok + Video Proxy

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Threads, Instagram, and TikTok adapters plus a signed streaming video proxy (`/v/`), with honest degrade paths and externalized version-fragile constants.

**Architecture:** Three more `PlatformAdapter` factories (same injected-`fetchFn` pattern). Video whose CDN URL needs upstream-specific headers (TikTok always; Threads/IG when direct playback fails) is declared by the adapter via a new `video.proxyHeaders` hint; the app wraps such URLs in a **short-lived HMAC-signed `/v/<token>` URL** at render time (never cached — tokens expire). The `/v/` route verifies the token, enforces a host allowlist (SSRF guard), forwards the client's `Range`, and streams the upstream body back (200/206) with a concurrency cap, per-request byte ceiling, and first-byte timeout. No full-body caching — Discord's media proxy already caches per-video.

**Tech Stack:** unchanged (Bun, Hono, TS strict, `bun test`). HMAC via Bun's Web Crypto (`crypto.subtle`), no new deps.

**Research basis:** `.superpowers/sdd/m4-research.md` — request shapes, response field paths, pinned constants, confidence ratings, and the datacenter-IP reality (Instagram will often fail from a bare VPS; degrade paths are first-class, not afterthoughts).

## Global Constraints

- Runtime deps: **hono only**. No live network in `bun test`.
- Adapter `match()` compares `url.hostname` exact-equality against a Set — never substrings.
- Adapters throw on failure → resolver degrades. The three M4 adapters additionally return an **informative text-only embed** (`kind: "link"`, explaining the block) rather than throwing when the platform is reachable but refuses data (login wall / empty post), so the user sees *why* instead of a bare redirect. Reserve `throw` for transport errors and genuinely-unavailable posts.
- Version-fragile constants (doc_ids, app-ids, LSD, friendly-names, mobile-API host/device ids, UAs) are **env-overridable with pinned defaults** (import defaults, `||`-fallback so blank `.env` values use them — the pattern established in M3).
- Signed media URLs must not outlive their usefulness: adapters set `ttlSeconds` on results with expiring media; `/v/` tokens carry their own short expiry independent of the metadata cache.
- Commit after every task; **no Claude commit attribution** (per CLAUDE.md).
- Shared platform UA lives in `src/lib/http.ts` (`PLATFORM_UA`); M4 adds browser/mobile UA constants there too.
- Test fetch fakes use the approved `as unknown as FetchFn` double-cast.

`src/lib/proxy-sign.ts`:
```ts
export interface ProxyPayload {
  url: string;
  headers: Record<string, string>;
  exp: number; // epoch ms
}

function b64urlEncode(bytes: Uint8Array): string {
  let bin = "";
  for (const b of bytes) bin += String.fromCharCode(b);
  return btoa(bin).replaceAll("+", "-").replaceAll("/", "_").replaceAll("=", "");
}

function b64urlDecode(s: string): Uint8Array {
  const pad = s.length % 4 === 0 ? "" : "=".repeat(4 - (s.length % 4));
  const bin = atob(s.replaceAll("-", "+").replaceAll("_", "/") + pad);
  const out = new Uint8Array(bin.length);
  for (let i = 0; i < bin.length; i++) out[i] = bin.charCodeAt(i);
  return out;
}

async function hmac(secret: string, data: string): Promise<Uint8Array> {
  const key = await crypto.subtle.importKey(
    "raw",
    new TextEncoder().encode(secret),
    { name: "HMAC", hash: "SHA-256" },
    false,
    ["sign"],
  );
  const sig = await crypto.subtle.sign("HMAC", key, new TextEncoder().encode(data));
  return new Uint8Array(sig);
}

function timingSafeEqual(a: Uint8Array, b: Uint8Array): boolean {
  if (a.length !== b.length) return false;
  let diff = 0;
  for (let i = 0; i < a.length; i++) diff |= a[i]! ^ b[i]!;
  return diff === 0;
}

export async function signProxyToken(secret: string, payload: ProxyPayload): Promise<string> {
  const body = b64urlEncode(new TextEncoder().encode(JSON.stringify(payload)));
  const sig = b64urlEncode(await hmac(secret, body));
  return `${body}.${sig}`;
}

export async function verifyProxyToken(
  secret: string,
  token: string,
  now: number,
): Promise<ProxyPayload | null> {
  const parts = token.split(".");
  if (parts.length !== 2) return null;
  const [body, sig] = parts as [string, string];
  let expected: Uint8Array;
  try {
    expected = await hmac(secret, body);
  } catch {
    return null;
  }
  let given: Uint8Array;
  try {
    given = b64urlDecode(sig);
  } catch {
    return null;
  }
  if (!timingSafeEqual(expected, given)) return null;
  try {
    const payload = JSON.parse(new TextDecoder().decode(b64urlDecode(body))) as ProxyPayload;
    if (typeof payload.exp !== "number" || now >= payload.exp) return null;
    if (typeof payload.url !== "string" || typeof payload.headers !== "object") return null;
    return payload;
  } catch {
    return null;
  }
}
```

### Task 24: Video proxy route (`/v/:token`)

**Files:**
- Create: `src/proxy.ts` (route factory)
- Modify: `src/app.ts` (mount), `src/lib/config.ts` (proxy config)

**Interfaces:**
- Config additions: `proxySecret: string` (env `PROXY_SECRET`, default `""` — when empty, proxying is disabled and adapters' proxied videos fall back to omitting video; a startup warn fires), `proxyHostAllowlist: string[]` (env `PROXY_HOST_ALLOWLIST` comma-sep, default the built-in list below), `proxyMaxConcurrent: number` (env, default 32), `proxyMaxBytes: number` (env, default 104857600 = 100 MB), `proxyTimeoutMs: number` (env, default 10000).
- Produces: `mountProxy(app: Hono, deps: { config: Config; logger: Logger; now?: () => number; fetchFn?: FetchFn }): void` registering `GET /v/:token`.
- Built-in allowlist (suffix match on hostname): `["cdninstagram.com", "fbcdn.net", "tiktokcdn.com", "tiktokcdn-us.com", "tiktokv.com", "tiktok.com", "muscdn.com", "byteoversea.com", "video.twimg.com"]`.

Behavior:
1. `verifyProxyToken(config.proxySecret, token, now())` → 404 `not found` on null.
2. Host allowlist: parse `payload.url`; if its hostname doesn't end with an allowed suffix → 403. (SSRF guard — a signed token can still only ever have been minted by us, but defense-in-depth.)
3. Concurrency: module-level in-flight counter; if `>= proxyMaxConcurrent` → 503 `busy`.
4. Fetch `payload.url` with `payload.headers` merged with a forwarded `Range` (from the client request, if present), `redirect: "follow"`, `signal: AbortSignal.timeout(config.proxyTimeoutMs)`. On throw/≥400 → 502.
5. Stream back: status (200/206) preserved; copy through `content-type`, `content-length`, `content-range`, `accept-ranges`, `etag`, `last-modified`; set `accept-ranges: bytes` and `cache-control: public, max-age=3600`; **drop** upstream `set-cookie`/auth headers. Enforce `proxyMaxBytes`: if a non-Range response advertises `content-length > proxyMaxBytes` → 502 (avoid unbounded); Range responses stream as-is.
6. Always decrement the in-flight counter (finally).

`src/proxy.ts`:
```ts
import type { Hono } from "hono";
import type { Config } from "./lib/config";
import type { Logger } from "./lib/logger";
import type { FetchFn } from "./adapters/types";
import { verifyProxyToken } from "./lib/proxy-sign";

const HOP_BY_HOP = new Set(["set-cookie", "authorization", "connection", "transfer-encoding"]);
const PASS_THROUGH = ["content-type", "content-length", "content-range", "accept-ranges", "etag", "last-modified"];

export function mountProxy(
  app: Hono,
  deps: { config: Config; logger: Logger; now?: () => number; fetchFn?: FetchFn },
): void {
  const { config, logger } = deps;
  const now = deps.now ?? Date.now;
  const fetchFn = deps.fetchFn ?? fetch;
  let inflight = 0;

  function hostAllowed(host: string): boolean {
    const h = host.toLowerCase();
    return config.proxyHostAllowlist.some((suffix) => h === suffix || h.endsWith(`.${suffix}`));
  }

  app.get("/v/:token", async (c) => {
    if (!config.proxySecret) return c.text("proxy disabled", 404);
    const payload = await verifyProxyToken(config.proxySecret, c.req.param("token"), now());
    if (!payload) return c.text("not found", 404);

    let target: URL;
    try {
      target = new URL(payload.url);
    } catch {
      return c.text("bad target", 400);
    }
    if (!hostAllowed(target.hostname)) return c.text("forbidden", 403);

    if (inflight >= config.proxyMaxConcurrent) return c.text("busy", 503);
    inflight++;
    try {
      const fwd = new Headers(payload.headers);
      const range = c.req.header("Range");
      if (range) fwd.set("Range", range);

      const up = await fetchFn(target.href, {
        headers: fwd,
        redirect: "follow",
        signal: AbortSignal.timeout(config.proxyTimeoutMs),
      });
      if (!up.ok && up.status !== 206) {
        logger.warn({ host: target.hostname, status: up.status }, "proxy upstream error");
        return c.text("bad gateway", 502);
      }
      const len = up.headers.get("content-length");
      if (up.status === 200 && len && Number(len) > config.proxyMaxBytes) {
        return c.text("bad gateway", 502);
      }
      const h = new Headers();
      for (const k of PASS_THROUGH) {
        const v = up.headers.get(k);
        if (v && !HOP_BY_HOP.has(k)) h.set(k, v);
      }
      h.set("accept-ranges", "bytes");
      h.set("cache-control", "public, max-age=3600");
      return new Response(up.body, { status: up.status, headers: h });
    } catch (err) {
      logger.warn({ host: target.hostname, err: String(err) }, "proxy fetch failed");
      return c.text("bad gateway", 502);
    } finally {
      inflight--;
    }
  });
}
```
